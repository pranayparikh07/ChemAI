        - Edge properties: weight, score, evidence, publication
        Output file: property_schema.txt

  [ ] 1.4 - Finalize ontology structure
        - Entity hierarchy
        - Property types and ranges
        - Constraint definitions
        Output file: ontology_definition.txt

TASK 2: KNOWLEDGE GRAPH DATA INTEGRATION
================================================================================
Status: [ ] Not Started

Sub-tasks:
  [ ] 2.1 - Extract MOLECULAR DATA
        - From ChEMBL: Canonical SMILES, ChEMBL ID, synonyms
        - From Vishwa's preprocessed data: Properties, classifications
        - Create Molecule nodes
        Output file: molecule_extraction.py
        Data: molecules_kg.csv

  [ ] 2.2 - Extract PROTEIN/TARGET DATA
        - From ChEMBL: Target information, UniProt IDs
        - From UniProt API: Protein sequences, families
        - Create Protein nodes with metadata
        Output file: protein_extraction.py
        Data: proteins_kg.csv

  [ ] 2.3 - Extract BIOACTIVITY DATA
        - From ChEMBL: IC50, pIC50, assay data
        - Link molecules to proteins via bioactivity
        - Create edges: molecule-[TARGETS]-protein
        - Include confidence scores
        Output file: bioactivity_linking.py
        Data: bioactivity_edges.csv

  [ ] 2.4 - Extract PATHWAY DATA
        - From KEGG, Reactome, or pathway databases
        - Map proteins to pathways
        - Create: protein-[INVOLVED_IN]-pathway edges
        Output file: pathway_extraction.py
        Data: pathway_edges.csv

  [ ] 2.5 - Extract TOXICITY DATA
        - Structural alerts
        - Known toxicity patterns
        - Link to molecules: molecule-[HAS_TOXICITY]-alert
        Output file: toxicity_linking.py
        Data: toxicity_edges.csv

  [ ] 2.6 - Extract DISEASE ASSOCIATIONS
        - From ChEMBL: Disease indications
        - Map proteins to diseases
        - Create: protein-[ASSOCIATED_WITH]-disease edges
        Output file: disease_linking.py
        Data: disease_edges.csv

TASK 5: GRAPH EMBEDDINGS FOR REASONING

Status: [ ] Not Started

Sub-tasks:

        Output file: graph_embeddings.py
        Directory: graph_db/

        Output: embeddings.pkl

        - Cosine similarity between nodes

        Output file: embedding_similarity_search.py

        - Given molecule M, find similar molecules

        Output file: recommendation_engine.py
