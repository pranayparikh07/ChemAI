TASK 2: MOLECULAR REPRESENTATION ENGINEERING
================================================================================
Status: [ ] Not Started

Sub-tasks:
  [ ] 2.1 - Convert molecules to GRAPH representation
        - Atoms = nodes, Bonds = edges
        - Implement graph construction function
        - Support for directed/undirected graphs
        Output file: graph_constructor.py
        Directory: data/graph_representations/

  [ ] 2.2 - Extract ATOM-LEVEL features
        - Atom type (C, N, O, S, P, F, Cl, Br, I, etc.)
        - Valency (degree in molecule)
        - Hybridization (sp, sp2, sp3)
        - Formal charge
        - Partial charges (MMFF94, GASTEIGER)
        - Aromaticity
        Output file: atom_feature_extractor.py
        Directory: models/

  [ ] 2.3 - Extract BOND-LEVEL features
        - Bond type (single, double, triple, aromatic)
        - Bond order
        - Conjugation status
        - Rotatable/fixed classification
        Output file: bond_feature_extractor.py
        Directory: models/

  [ ] 2.4 - Implement fingerprint generation
        - Morgan fingerprints (2048 bits, already used)
        - RDKit descriptors (for ML models)
        - Graph-based representations
        Output file: fingerprint_generator.py
        Directory: models/

TASK 3: FEATURE ENGINEERING & CANONICAL FORMS
================================================================================
Status: [ ] Not Started

Sub-tasks:
  [ ] 3.1 - Canonical SMILES standardization
        - Implement SMILES canonicalization
        - Remove duplicates based on canonical form
        - Handle tautomers and stereoisomers
        Output file: smiles_standardizer.py
        Directory: scripts/

  [ ] 3.2 - Extract molecular PROPERTIES
        - Molecular Weight (MW)
        - Partition Coefficient (LogP)
        - Hydrogen Bond Donors (HBD)
        - Hydrogen Bond Acceptors (HBA)
        - Polar Surface Area (PSA)
        - Rotatable Bonds
        - Aromatic rings
        Output file: property_calculator.py
        Directory: models/

  [ ] 3.3 - Build FEATURE MATRIX
        - Create feature vectors for each molecule
        - Normalize/standardize features
        - Handle missing values
        - Feature scaling (StandardScaler, MinMaxScaler)
        Output file: feature_matrix_builder.py
        Directory: models/
        Output data: features.pkl, features.csv

  [ ] 3.4 - Drug-likeness metrics
        - Lipinski's Rule of 5
        - QED (Quantitative Estimation of Drug-likeness)
        - Synthetic Accessibility Score
        Output file: druglikeness_calculator.py
        Directory: models/
